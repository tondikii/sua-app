# UX & Copywriting Notes

## Core Philosophy
The codebase is in English (e.g., `Itinerary`, `Wishlist`), but the UI is entirely in **Indonesian**. The tone must be warm, playful, and conversational ("Bahasa Gaul / Bahasa Sehari-hari"). It should feel like a partner talking to you, not a machine.

## Terminology Mapping (Code -> UI)

| Technical/Code Name | Indonesian UI Label | Context / Vibe |
| :--- | :--- | :--- |
| `Explore` | **Eksplor** atau **Cari Ide** | Inviting, exciting. |
| `Wishlist` | **Wishlist Kita** | Emphasizes shared ownership. |
| `Itinerary` | **Agenda Sua** / **Rencana Kita** | Formal enough to be a plan, soft enough to be romantic. |
| `Add Partner` | **Ajak Pasangan** | Friendly, not technical. |
| `WFC` | **WFC-an** | Very natural Indonesian slang. |

## Urgency Levels (Wishlist Sorting)
*This is a core feature. The copy here must be exactly as follows:*
1. 🔥 **Pengen banget-bangett** (Crucial: use double 't' for emphasis).
2. ✨ **Pengen bangett**.
3. 👀 **Pengen ajaa**.

## Empty States
Empty states should encourage action without being annoying.
* **Empty Agenda**: *"Wah, masih wacana doang nih? Yuk susun jadwal kencanmu sekarang!"*
* **Empty Wishlist**: *"Belum ada tempat inceran nih. Coba eksplor ide kencan seru dulu yuk!"*
* **No Partner Linked**: *"Sendirian aja nih? Ajak ayang pakai Sua biar kencannya makin gampang!"*
